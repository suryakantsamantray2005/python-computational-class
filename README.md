# python-computational-class
Python class-based project containing mathematical utility functions such as factorial, prime checking, divisors, and multiplication tables.

This project is a beginner-level Python program using Object-Oriented Programming (OOP).

The `Computational` class contains different mathematical utility functions such as:

- Factorial calculation
- Sum of natural numbers
- Prime number checking
- Co-prime checking
- Multiplication table generation
- Divisor listing
- Prime divisor listing

## Features

### Factorial
Calculates the factorial of a number.

### Natural Number Sum
Finds the sum of first `n` natural numbers.

### Prime Number Test
Checks whether a number is prime or not.

### Co-Prime Test
Checks whether two numbers are co-prime.

### Multiplication Table
Generates multiplication table up to 10.

### Divisor List
Returns all divisors of a number.

### Prime Divisors
Returns only prime divisors of a number.

## Technologies Used

- Python 3
- Object-Oriented Programming (OOP)
